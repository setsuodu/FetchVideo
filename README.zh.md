# FetchVideo
Docker Services &amp; CrossPlatfom App

# 客户端

是一个Windows的控制台程序, 按照以下步骤下载视频:
1. 双击启动程序 FetchVideo.exe;
2. 输入/粘贴 视频地址;
3. 按下键盘 '回车';
4. 返回桌面查看下载的 .mp4 文件;
